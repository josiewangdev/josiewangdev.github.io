1. IBInspectable Value

Setting the varible with @IBInspectable allows us to edit it in xib file.

@IBInspectable var cornerRadius: CGFloat = 0 {
    didSet {
        layer.cornerRadius = cornerRadius
        layer.masksToBounds = cornerRadius > 0
    }
}
